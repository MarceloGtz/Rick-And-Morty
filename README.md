## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceloGtz/Rick-And-Morty.git
```

Go to the project directory

```bash
  cd Rick-And-Morty
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
